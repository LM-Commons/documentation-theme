## Requirements

- PHP 8.2 or higher

## Installation

*package-repo* only officially supports installation through Composer.

Install the module:

```sh
$ composer require lm-commons/package-repo
```

You will be prompted by the Laminas Component Installer plugin to inject the
component into your `config/config.php` file.
